lowellhs.org

This website provides free subdomains for Lowell High School students.

To claim your subdomain, follow these instructions:
  1. Fork lowellhs.org and clone it locally.
  2. Add your host to hosts.json
    [ ..., { "HostName": "your-subdomain", "RecordType": "CNAME", "Address": "website-url.com.", } ]
  3. Run node sync.js to sort and format the host file.
  4. Open a pull request with some basic information. When it is merged, Travis CI will update the hosts on Namecheap.
