This website provides free subdomains for Lowell High School students.

To claim your subdomain, follow these instructions:
  1. Fork lowellhs.org and clone it locally.
  2. Add your host to hosts.json { ..., { "HostName": "your-subdomain", "RecordType": "CNAME", "Address": "yourname.github.io.", "TTL": 0 } }
  3. Run node sync.js to sort and format the host file.
  4. Open a pull request with some basic information. I'll merge it and Travis CI will update the hosts on Namecheap.
